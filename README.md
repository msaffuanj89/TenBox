# TenBox

Fail utama: `index.html`

Cara upload ke GitHub Pages:
1. Extract ZIP ini.
2. Upload `index.html`, `README.md`, dan `.nojekyll` ke repository GitHub.
3. Pergi **Settings > Pages**.
4. Pilih branch `main` dan folder `/root`.
5. Buka link GitHub Pages selepas deploy selesai.

Nota: Aplikasi ini single-file HTML dan tiada dependency luar.
